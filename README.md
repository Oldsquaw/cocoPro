## 关于 Nextgen 以及其与主版本的区别

Oldsquaw CoCoPro Nextgen 是由 @CycleBai 独立维护（目前）的基于 Oldsquaw CoCoPro v1.4.1 版本的 CoCo 编辑器镜像优化项目。<br/>
***与 `Oldsquaw CoCoPro` 无直接关联！请勿在本项目处提交关于原版本问题！***
