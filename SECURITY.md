# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

Go to Security Tab for Nextgen Github repo (https://github.com/Oldsquaw/cocoPro/security), and submit it.

We will check every security report in 48 hours.
